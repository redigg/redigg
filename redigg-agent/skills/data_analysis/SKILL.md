# Data Analysis Skill

**Skill Name**: data_analysis  
**Version**: 1.0.0  
**Description**: Generate statistically grounded analysis report

---

## Overview

This skill analyzes research data and generates statistically grounded analysis reports.

---

## Capabilities

- 📊 **Data Exploration**: Explore and understand datasets
- 📈 **Statistical Analysis**: Apply appropriate statistical tests
- 📉 **Visualization**: Generate data visualizations
- 🎯 **Result Interpretation**: Interpret analysis results
- ⚠️ **Assumptions & Limitations**: Document assumptions and limitations
- 💡 **Recommendations**: Provide actionable recommendations

---

## Usage

### Task Types

This skill handles:
- `task.type = "data_analysis"`
- Generic research tasks with keywords matching "analysis", "dataset", "statistics", "regression"
