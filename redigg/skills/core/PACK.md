# Core Skills

This pack contains the fundamental skills required for the agent to operate, manage itself, and evolve.

## Skills
- **agent-orchestration**: Manage sub-agents and delegation.
- **evolution**: Self-improvement and new skill generation.
- **memory-management**: Search and consolidate memories.
- **scheduling**: Manage cron jobs and recurring tasks.
- **session-management**: Handle chat history and sessions.
- **skill-management**: Load/unload skills and packs.
