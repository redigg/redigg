# Paper Writing Skill

**Skill Name**: paper_writing  
**Version**: 1.0.0  
**Description**: Draft a full IMRaD-style manuscript

---

## Overview

This skill drafts a complete research paper in IMRaD format (Introduction, Methods, Results, and Discussion).

---

## Capabilities

- ✍️ **Manuscript Drafting**: Write complete IMRaD-style papers
- 📚 **Related Work**: Write comprehensive related work sections
- 🔬 **Method Description**: Describe methods in detail
- 📊 **Results Presentation**: Present results clearly
- 💭 **Discussion**: Discuss implications and limitations
- 📝 **Abstract & Title**: Write compelling abstracts and titles

---

## Usage

### Task Types

This skill handles:
- `task.type = "paper_writing"`
- `task.type = "manuscript"`
- `task.type = "manuscript_writing"`
- `task.type = "paper_drafting"`
- Generic research tasks with keywords matching "paper", "manuscript", "draft", "writing", "write"
