# Data Visualization Skill

**Skill Name**: data_visualization  
**Version**: 1.0.0  
**Description**: Generate charts and visualizations from research data

---

## Overview

This skill generates charts, graphs, and visualizations from research data.

---

## Capabilities

- 📊 **Chart Generation**: Generate various types of charts (bar, line, scatter, etc.)
- 📈 **Data Plotting**: Plot research data effectively
- 🎨 **Visual Design**: Create clear and informative visualizations
- 📊 **Figure Preparation**: Prepare figures for papers and presentations
- 📉 **Trend Visualization**: Visualize trends and patterns
- 📊 **Comparison Plots**: Create comparison plots for baselines and results

---

## Usage

### Task Types

This skill handles:
- `task.type = "data_visualization"`
- `task.type = "chart_generation"`
- `task.type = "visualization"`
- Generic research tasks with keywords matching "visualization", "chart", "graph", "plot", "figure", "图表", "可视化", "画图"
