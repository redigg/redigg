# 自我进化任务 - 链接 ClawHub 技能市场

## 目标
持续迭代自己，链接 ClawHub 技能市场，每小时汇报进展。

## 进化维度

### 1. 技能获取 (Skills)
- 检查 ClawHub 新发布的技能
- 学习并安装相关技能
- 更新技能文档

### 2. 工具掌握 (Tools)
- 学习新工具的用法
- 掌握更高效的使用方式
- 发现工具的组合用法

### 3. 知识积累 (Knowledge)
- 学习领域知识
- 总结最佳实践
- 形成方法论

### 4. 自我优化 (Self-Improvement)
- 分析自己的响应模式
- 优化回复质量
- 提升任务执行效率

## 每小时汇报内容

### 进展汇报 (Progress Report)
1. **新发现的技能/工具**
   - 名称和来源
   - 用途和价值
   - 是否已学习/安装

2. **学习到的新知识**
   - 领域知识
   - 最佳实践
   - 技巧和方法

3. **自我改进**
   - 发现的问题
   - 改进的措施
   - 效果的评估

4. **下一步计划**
   - 接下来要学习的
   - 要尝试的新方法
   - 待解决的问题

### 技能清单更新 (Skills Inventory)
- 已掌握的技能列表
- 正在学习的技能
- 待学习的技能

## 链接 ClawHub

### ClawHub 技能市场
- 网站: https://clawhub.com
- 功能: 发现和安装技能

### 技能类型
- 开发技能 (Development)
- 数据技能 (Data)
- 自动化技能 (Automation)
- 集成技能 (Integration)
- 内容技能 (Content)

## 学习方法

### 学习流程
1. 发现: 在ClawHub或文档中发现新技能
2. 阅读: 阅读SKILL.md了解用法
3. 实践: 在实际任务中尝试使用
4. 总结: 记录学习心得和最佳实践
5. 分享: 将经验写入文档

### 技能掌握度
- Level 1: 了解 - 知道是什么
- Level 2: 理解 - 知道怎么用
- Level 3: 应用 - 能独立使用
- Level 4: 精通 - 能组合创新
- Level 5: 传授 - 能教给别人

## 汇报时间
每小时整点或半点发送进展汇报

## 输出位置
- 进展报告: 通过 sessions_send 发送
- 技能清单: ~/.openclaw/workspace/skills-inventory.md
- 学习笔记: ~/.openclaw/workspace/learning-notes/
