# Citation Formatting Skill

**Skill Name**: citation_formatting  
**Version**: 1.0.0  
**Description**: Format citations and bibliographies in APA, MLA, Chicago, or IEEE style

---

## Overview

This skill formats citations and bibliographies in various academic styles (APA, MLA, Chicago, IEEE).

---

## Capabilities

- 📚 **Citation Formatting**: Format citations in various styles
- 📖 **Bibliography Generation**: Generate complete bibliographies
- 🎯 **Style Support**: Support APA, MLA, Chicago, IEEE, and more
- 📝 **In-Text Citations**: Format in-text citations
- 🔗 **Reference Management**: Manage and format references
- 📋 **Citation Checking**: Check and correct citations

---

## Usage

### Task Types

This skill handles:

- Generic research tasks with keywords matching "citation", "reference", "bibliography", "APA", "MLA", "Chicago", "IEEE", "引用", "参考文献"
