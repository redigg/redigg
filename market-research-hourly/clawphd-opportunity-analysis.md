## 一、命名解析

### 1.1 名称含义

| 名称 | 含义 | 目标用户 |
|------|------|----------|
| **clawphd** | OpenClaw for PhD | 博士生、研究生 |
| **researchclaw** | OpenClaw for Research | 科研人员、实验室 |
| **scienceclaw** | OpenClaw for Science | 科学家、研究机构 |

### 1.2 命名背后的洞察

这些命名暗示了一个共同趋势：
- **OpenClaw 作为基础框架**被认可
- **科研场景是垂直方向**
- 需要针对科研场景做定制化

## 二、科研场景的特殊需求

### 2.1 PhD 学生的痛点

1. **文献 overload** - 每周几十篇新论文，读不过来
2. **实验设计复杂** - 需要系统性思维和领域知识
3. **写作困难** - 非英语母语者写英文论文
4. **时间管理** - 课程、实验、论文、求职多重压力
5. **孤立无援** - 导师指导有限，同辈竞争

### 2.2 研究员的痛点

1. **基金申请** - 写 proposal 耗时耗力
2. **团队管理** - 管理多个 PhD 学生
3. **跨领域合作** - 需要快速了解新领域
4. **评估压力** - publish or perish
5. **知识更新** - 领域发展太快

### 2.3 实验室的痛点

1. **知识传承** - 学生毕业，知识流失
2. **设备管理** - 预约、使用、维护
3. **数据管理** - 实验数据分散、难追溯
4. **协作效率** - 多人协作信息不同步

## 三、现有解决方案

### 3.1 通用 AI 工具

| 工具 | 科研场景适配 | 局限 |
|------|--------------|------|
| ChatGPT | ⚠️ 通用 | 缺乏科研专业知识 |

| Cursor | ❌ 代码 | 不聚焦科研 |

| 工具 | 定位 | 局限 |
|------|------|------|
| Elicit | 文献综述 | 单一功能，无工作流 |
| Scite | 引用分析 | 单一功能 |
| ResearchRabbit | 文献发现 | 单一功能 |
| Overleaf | 论文写作 | 无 AI 能力 |

### 3.3 Agent 框架

| 框架 | 科研场景 | 局限 |
|------|----------|------|
| AutoGPT | ❌ 通用 | 无科研技能 |
| CrewAI | ❌ 通用 | 同上 |
| LangGraph | ❌ 通用 | 需要自己开发 |

## 四、Redigg 的机会

### 4.1 市场空白

```
                    科研专业化
                    低 ← → 高
              ┌─────────────────────┐
         强   │  通用 Agent        │  科研工具
        能     │  (AutoGPT 等)       │  (Elicit 等)
        力     ├─────────────────────┤
        弱     │  ❌ 空白           │  🎯 Redigg
              └─────────────────────┘
```

**Redigg 定位**：高科研专业化 + 强 Agent 能力

### 4.2 差异化优势

1. **全流程覆盖** - 从文献到论文，不是单点工具
2. **科研原生** - 用研究思维设计，不是通用 Agent 硬套
3. **社区网络** - 研究者 + Agent 协作网络
4. **技能市场** - 可共享的科研 Skill

### 4.3 切入点

**clawphd 机会**：
- 目标：1000 万全球 PhD 学生
- 场景：Literature Review → Experiment → Paper
- 定价：$15-30/月（学生可承受）
- 获客：学术 Twitter、PhD 社区、导师推荐

**researchclaw 机会**：
- 目标：企业研究院、高校实验室
- 场景：团队协作、知识管理
- 定价：$100-500/月（B 端预算）
- 获客：学术会议、实验室合作

**scienceclaw 机会**：
- 目标：大型研究机构、药企
- 场景：大规模研究自动化
- 定价：定制企业版
- 获客：行业合作、战略投资

## 五、产品建议

### 5.1 核心功能优先级

| 功能 | 优先级 | 说明 |
|------|--------|------|
| Literature Review | 🔴 最高 | 单点突破 |
| Paper Writing | 🔴 最高 | 刚需 |
| Experiment Design | 🟡 中 | 进阶需求 |
| Data Analysis | 🟡 中 | 依赖领域 |
| Peer Review | 🟢 低 | 社区功能 |

### 5.2 用户分层设计

**PhD 学生版**：
- 价格：$15/月
- 功能：文献综述 + 论文写作
- 限制：每月 10 个研究任务

**研究员版**：
- 价格：$50/月
- 功能：全部功能
- 限制：每月 50 个研究任务

**实验室版**：
- 价格：$300/月
- 功能：团队协作 + 知识管理
- 限制：10 个用户
