## 🔍 关键发现和洞察

### Elicit API 深度分析

**API 核心信息**：
| 项目 | 详情 |
|------|------|
| API 版本 | 0.0.1（OpenAPI 3.1.0） |
| 论文库规模 | 1.25 亿 + 学术论文 |
| 访问要求 | Pro 计划及以上 |
| API 费用 | 无额外费用（计入工作流配额） |
| 速率限制 | Pro: 100 篇/请求，100 请求/天；Team: 200/200；Enterprise: 5000/无限制 |

**核心端点**：
1. **POST /api/v1/search**: 语义搜索 1.25 亿论文
   - 支持年份、关键词、研究类型、期刊分区、PDF 可用性等过滤
   - 返回：title/authors/year/abstract/doi/pmid/venue/citedByCount/urls

2. **POST /api/v1/reports**: 异步生成研究简报
   - 处理时间：5-15 分钟
   - 支持自定义搜索论文数（默认 50）和提取论文数（默认 10）
   - 完成后提供 pdfUrl/docxUrl 下载

**开发者生态**：
- 完整示例库：https://github.com/elicit/api-examples
- 多语言支持：curl/JavaScript/Python
- 现成集成：CLI 工具、Slack 机器人、Claude Code 技能
- **关键洞察**: Claude Code 集成直接竞争 Redigg 的 Agent 能力
