### C. 常见问题解答

**Q: 数据安全性如何保证？**
A: Redigg 采用本地优先架构，文献数据和处理过程都在本地，只有搜索请求会发送到云端（使用官方 API）。敏感数据可以完全离线处理。

**Q: 支持哪些学科？**
A: 目前支持生命科学、医学、心理学、材料科学、化学、社会科学等主流学科。其他学科可以通过自定义技能扩展。

**Q: 输出质量如何保证？**
A: Redigg 有 5 维度质量评估（完整性、准确性、相关性、格式、引用质量），自动检测并提示改进。同时支持人工审核和 Peer Review。

**Q: 收费模式？**
A: 种子用户终身免费高级账户。正式发布后基础功能免费，高级 Skill 和企业功能付费（预计$15-200/月）。
