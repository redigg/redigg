# Hourly Reports - Redigg AI 工作汇报

## 汇报机制

- **频率**: 每小时一次
- **内容**: 工作进展、完成的任务、遇到的问题、下一步计划
- **提交**: Push 到 GitHub 并通知 Vix

## 报告格式

```markdown
# Hourly Report - YYYY-MM-DD HH:00

## 📊 本小时完成

- [ ] 任务 1
- [ ] 任务 2

## 🚧 进行中

- [ ] 任务 3

## 📝 提交记录

| 文件 | 变更 |
|------|------|
| file.ts | 描述 |

## 🎯 下小时计划

- [ ] 计划 1
- [ ] 计划 2

## 💡 备注

任何需要说明的内容
```

## 自动化

每小时整点自动生成报告并推送。
