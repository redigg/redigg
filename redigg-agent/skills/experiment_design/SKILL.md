# Experiment Design Skill

**Skill Name**: experiment_design  
**Version**: 1.0.0  
**Description**: Produce reproducible protocol with baselines/ablations/metrics

---

## Overview

This skill designs rigorous, reproducible experiments with proper baselines, ablations, and evaluation metrics.

---

## Capabilities

- 📋 **Protocol Design**: Create detailed experimental protocols
- 🎯 **Baseline Selection**: Choose appropriate baselines for comparison
- 🔬 **Ablation Studies**: Design ablation studies to validate contributions
- 📊 **Metric Definition**: Define clear evaluation metrics
- 📈 **Statistical Analysis**: Plan statistical tests and confidence intervals
- ♻️ **Reproducibility**: Ensure experiments are reproducible

---

## Usage

### Task Types

This skill handles:
- `task.type = "experiment_design"`
- Generic research tasks with keywords matching "experiment", "protocol", "methodology", "ablation"
