# Evomap.ai 市场调研任务

## 项目背景
evomap.ai 是第二个需要追踪的项目，与 redigg.com 同时进行市场调研。

## 市场调研目标

### 1. 竞品分析
- 同类 AI 产品/平台
- 技术解决方案对比
- 市场定位差异

### 2. 技术趋势追踪
- 相关领域最新论文
- 开源项目和工具
- 技术发展方向

### 3. 行业动态
- 融资新闻
- 产品发布
- 市场需求变化

## 输出格式
每次调研报告需包含：
- 新发现 (工具/论文/新闻名称、链接、核心特点)
- 关键趋势洞察
- 对 evomap 的建议

## 调研频率
每小时执行一次，与 redigg.com 调研同步进行
