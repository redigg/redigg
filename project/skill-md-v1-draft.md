## 一、概述

skill.md是Redigg平台中研究技能的结构化文档标准。它定义了研究技能的接口、行为和元数据，使得：

1. **标准化**: 不同开发者创建的skill可以互操作
2. **可组合**: 多个skill可以组合成复杂的研究工作流
3. **可发现**: 用户可以方便地发现和理解skill的功能
4. **可交易**: skill可以在市场上交易

## 三、字段详细说明

### 3.1 元数据部分

#### Skill ID
- **格式**: `skill-[领域]-[功能名]-[版本]`
- **示例**: `skill-literature-review-v1`
- **要求**: 全局唯一，使用小写字母、连字符分隔

#### 版本
- **格式**: 语义化版本号（MAJOR.MINOR.PATCH）
- **示例**: `1.0.0`
- **说明**:
  - MAJOR: 不兼容的API变更
  - MINOR: 向下兼容的功能新增
  - PATCH: 向下兼容的问题修复

#### 许可证
- **推荐**: MIT、Apache-2.0、GPL-3.0等
- **说明**: 明确skill的使用许可

### 3.2 功能说明部分

#### 输入参数
- **类型**: string、number、boolean、array、object
- **必填**: 是/否
- **描述**: 清晰说明参数的用途和格式

#### 输出格式
- **类型**: string、number、boolean、array、object
- **描述**: 清晰说明输出字段的含义
