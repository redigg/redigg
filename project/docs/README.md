# Redigg 开发群

这个群用于记录 Redigg 开发的相关讨论和工作。

## 群信息

**项目名称**: redigg develop

**对应项目**: `./project/`

**推进频率**: Hourly（每小时）

---

## 核心定位

**研究平台** - 与 evowork（工作平台）形成互补。

---

## 与 evowork 的区别

- **redigg**：研究平台
- **evowork**：工作平台（AI Agent 自进化工作平台）

---

## 核心任务

### 1. 市场调研
- AI Agent 工作流相关调研
- 研究协作平台调研
- 众包平台调研

### 2. 竞品分析
- 追踪竞品平台动态
- 分析竞品功能和特点
- 找出差异化机会

### 3. 产品开发
- Redigg 平台功能开发
- 用户体验优化
- 技术迭代

## 工作流程

### 每小时 Review 内容

1. **开发/调研进展回顾** - 过去一小时完成的工作
2. **竞品动态追踪** - 相关平台的新动态
3. **技能评估** - 当前使用的技能，哪些需要优化
4. **最佳实践** - 发现的好方法、好模式
5. **待办事项** - 下一小时需要做的事情
