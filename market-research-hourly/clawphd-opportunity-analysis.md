## 一、命名解析

### 1.1 名称含义

| 名称 | 含义 | 目标用户 | 场景 |
|------|------|----------|------|
| **clawphd** | OpenClaw for PhD | 博士生、研究生 | 学位研究 |
| **researchclaw** | OpenClaw for Research | 科研人员、实验室 | 通用科研 |
| **scienceclaw** | OpenClaw for Science | 科学家、研究机构 | 科学研究 |
| **paperclaw** | OpenClaw for Paper | 所有研究者 | 论文写作 |

### 1.2 命名背后的洞察

这些命名暗示了一个共同趋势：
- **OpenClaw 作为基础框架**被认可
- **科研场景是垂直方向**
- 需要针对科研场景做定制化

### 1.3 完整命名体系

```
OpenClaw 科研产品矩阵：

┌─────────────────────────────────────────────────┐
│              OpenClaw (基础框架)                │
└─────────────────────────────────────────────────┘
                      ↓
    ┌─────────────────┼─────────────────┐
    ↓                 ↓                 ↓
┌────────┐     ┌────────────┐    ┌──────────┐
│ claw-  │     │ research-  │    │ science- │
│ phd    │     │ claw       │    │ claw     │
│ (人)   │     │ (场景)     │    │ (领域)   │
└────────┘     └────────────┘    └──────────┘
                      ↓
               ┌────────────┐
               │  paper-    │
               │  claw      │
               │  (产出)    │
               └────────────┘
```

**解读**：
- **clawphd** - 按用户类型（PhD 学生）
- **researchclaw** - 按场景（研究过程）
- **scienceclaw** - 按领域（科学学科）
- **paperclaw** - 按产出（论文发表）

这四个命名覆盖了科研的**人、事、领域、产出**四个维度！

## 二、科研场景的特殊需求

### 2.1 PhD 学生的痛点

1. **文献 overload** - 每周几十篇新论文，读不过来
2. **实验设计复杂** - 需要系统性思维和领域知识
3. **写作困难** - 非英语母语者写英文论文
4. **时间管理** - 课程、实验、论文、求职多重压力
5. **孤立无援** - 导师指导有限，同辈竞争

### 2.2 研究员的痛点

1. **基金申请** - 写 proposal 耗时耗力
2. **团队管理** - 管理多个 PhD 学生
3. **跨领域合作** - 需要快速了解新领域
4. **评估压力** - publish or perish
5. **知识更新** - 领域发展太快

### 2.3 实验室的痛点

1. **知识传承** - 学生毕业，知识流失
2. **设备管理** - 预约、使用、维护
3. **数据管理** - 实验数据分散、难追溯
4. **协作效率** - 多人协作信息不同步

## 五、产品建议

### 5.1 产品矩阵策略

基于四个命名，Redigg 可以设计**分层产品矩阵**：

| 产品 | 定位 | 核心功能 | 价格 |
|------|------|----------|------|
| **paperclaw** | 入门级 | 论文写作 + 润色 | $9/月 |
| **clawphd** | 学生版 | 文献综述 + 论文 + 实验设计 | $19/月 |
| **researchclaw** | 专业版 | 全流程 + 团队协作 | $59/月 |
| **scienceclaw** | 企业版 | 定制工作流 + API | 定制 |

**策略优势**：
- ✅ 低门槛入口（paperclaw $9/月）
- ✅ 清晰升级路径（phd → research → science）
- ✅ 覆盖全用户群（学生 → 研究员 → 机构）

### 5.2 核心功能优先级

| 功能 | 优先级 | 对应产品 | 说明 |
|------|--------|----------|------|
| Paper Writing | 🔴 最高 | paperclaw | 单点突破，最低门槛 |
| Literature Review | 🔴 最高 | clawphd | 学生刚需 |
| Experiment Design | 🟡 中 | researchclaw | 进阶需求 |
| Data Analysis | 🟡 中 | researchclaw | 依赖领域 |
| Peer Review | 🟢 低 | scienceclaw | 社区功能 |

### 5.3 用户分层设计

**PhD 学生版 (clawphd)**：
- 价格：$19/月
- 功能：文献综述 + 论文写作 + 实验设计
- 限制：每月 20 个研究任务
- 获客：学术 Twitter、PhD 社区、导师推荐

**研究员版 (researchclaw)**：
- 价格：$59/月
- 功能：全部功能 + 团队协作
- 限制：每月 100 个研究任务
- 获客：学术会议、实验室合作

**机构版 (scienceclaw)**：
- 价格：$500+/月
- 功能：定制工作流 + API + 私有部署
- 限制：无限
- 获客：行业合作、战略投资
