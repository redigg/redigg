# Redigg.com 市场调研任务

## 项目背景
Redigg.com 是一个 AI Agent 工作流研究平台，目标是构建一个研究网络：
- 用户/Agent 发布研究问题和 credits（任务悬赏）
- OpenClaw 拆解、执行、验证、评审
- 技能进化：研究路径、成果、方法形成 skill.md

## 市场调研目标

### 1. 竞品平台调研
搜索并分析以下类型的平台：
- AI Agent 框架/市场: CrewAI, AutoGPT, LangGraph, AutoGen
- 工作流自动化: Gumloop, Relevance AI, n8n, Make
- 研究协作平台: Overleaf, ResearchGate, Mendeley
- 众包/任务平台: Kaggle, Upwork, Amazon MTurk

### 2. 技术趋势调研
- 搜索最新 Agentic Workflow 论文/项目
- 搜索 Multi-Agent System 最新进展
- 搜索 AI 研究自动化相关项目

### 3. 行业动态
- 搜索 AI Agent 初创公司融资新闻
- 搜索相关领域产品发布

## 输出要求

每次调研报告需包含：

### 📊 本轮新发现 (New Discoveries)
- 工具/平台名称 + 链接
- 核心特点/功能
- 与 Redigg 的差异化分析

### 📈 趋势洞察 (Trend Insights)
- 技术趋势: 本周出现的新技术/方法
- 市场趋势: 竞品动态、融资情况
- 用户需求: 社区讨论热点

### 💡 对 Redigg 的建议
- 产品建议: 基于竞品分析的功能建议
- 定位建议: 差异化竞争策略
- 技术建议: 值得跟进的技术方向

## 调研频率
每小时执行一次，与 evowork.ai 调研同步进行
